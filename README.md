# Basic Login System

## Description
A simple login authentication system using PHP and MySQL.
It checks user credentials from the database and redirects to a dashboard on success.

## Technologies Used
- PHP
- MySQL
- XAMPP
- HTML

## Features
- Login form
- Database validation
- Redirect on success
- Error on failure
- Logout system
